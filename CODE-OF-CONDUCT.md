このプロジェクトでは、当コミュニティで期待される動作を明らかにするために共同作成者契約で定義された倫理規定を採用しています。

詳細については、「[.NET Foundation Code of Conduct](https://dotnetfoundation.org/code-of-conduct)」(.NET Foundation の倫理規定) を参照してください。
